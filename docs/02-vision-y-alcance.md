# VISIÓN Y ALCANCE

## Visión
Construir una plataforma modular y escalable que permita gestionar dispositivos de monitoreo ambiental,
analizar datos en tiempo real y emitir alertas ante condiciones de riesgo.

## Alcance actual
- Autenticación y autorización centralizada
- Gestión de dispositivos
- Procesamiento de alertas
- Registro de datos históricos
- Notificaciones básicas
- Orquestación completa con Docker

## Fuera de alcance (por ahora)
- Integración directa con hardware IoT
- Persistencia en bases de datos productivas
- Frontend web
