# C4 – CONTAINER DIAGRAM

## Containers

### Web Application
Interfaz de usuario para monitoreo y gestión.

### API Gateway
Punto único de entrada para clientes web.

### Auth Service
Gestión de identidad y roles.

### Device Service
Gestión de dispositivos IoT.

### Alert Service
Detección y gestión de eventos críticos.

### Data Service
Persistencia y análisis de datos históricos.

### Notification Service
Envío de alertas por canales externos.

### Message Broker
Recepción y distribución de eventos IoT.

## Communication
- HTTPS para clientes
- REST interno
- Eventos asincrónicos para IoT
